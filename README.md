# Fatigue_landmark

산업 현장 작업자의 피로도를 실시간으로 판단하기 위한 landmark 기반 피로도 탐지 코드입니다.  
카메라 영상에서 얼굴 및 자세 landmark를 추출한 뒤, 눈 감김, 하품, 자세 변화를 기반으로 피로도를 계산합니다.

## 프로젝트 개요
본 코드는 작업자의 상태를 비접촉 방식으로 분석하여 피로 징후를 조기에 탐지하는 것을 목표로 합니다.  
랜드마크 기반 특징 추출을 통해 다음 요소를 판단합니다.

- 눈 감김 정도
- 하품 여부
- 자세 변화
- 종합 피로도 점수

## 파일 구성

- `main.py` : 전체 실행 코드
- `config.py` : 임계값 및 주요 파라미터 설정
- `landmark_utils.py` : 랜드마크 처리 및 보조 함수
- `eye_model.py` : 눈 감김 기반 피로도 계산
- `yawn_model.py` : 하품 검출
- `posture_model.py` : 자세 기반 피로도 계산
- `fusion_model.py` : 각 피로 지표를 종합하여 최종 점수 산출

## 주요 기능

1. 얼굴 및 자세 landmark 기반 특징 추출
2. 눈 감김 지속 시간 분석
3. 하품 여부 판단
4. 자세 기울어짐 및 변화량 분석
5. 각 특징을 종합한 최종 피로도 점수 계산

## 활용 목적
본 코드는 산업 현장 내 작업자의 피로 상태를 실시간으로 파악하고,  
위험 상황을 사전에 감지하기 위한 시스템 개발 과정의 일부로 작성되었습니다.
